# Overview
A curated list of awesome 3D point cloud denoising (and closely related fields) papers.

This is a work in _continuous progress_.

For any suggestions, please contact garnungalejandro@gmail.com.

Thank you and enjoy the 3D point cloud denoising world.

**Updated: 02/01/2025**

<h2>Keywords</h2>

`trad.`: traditional filter | `var.`: variational-based | `pde`: PDE-based | `spec.`: spectral-based | `dl.`: deep learning | `un.`: unsupervised | `rec.`: resampling/compression | `sur.`: surface reconstruction  | `ne.`: normal estimation | `inp.`: inpainting | `res.`: super-resolution |  `mesh.`: mesh denoising  |  `oth.`: other
